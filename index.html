Introduction

The full guide to use LInjector. The open-source executor.

Requirements

  • WebView2 Runtime

  • .NET 4.8 Runtime

  • Microsoft Visual C++ Redistributables x86 or x64

Downloading LInjector

There are multiple ways to download LInjector, there's a list with the supported ones:

  • LInjector Releases (GitHub)

  • Website

  • Building from repo (not recommended)

False-positive detections (Antivirus)

I'm not going to say to disable your antivirus. You can add an exclusion to the folder where LInjector is located. Check: Add an exclusion to Windows Security.
