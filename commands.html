Команды

Функционал скриптов позволяет создавать свои подкоманды в .bots.

Создать свою команду очень просто:

Пример использования

function onCommand_test(args) {
    //Выполняется при вводе команды .bots test
    //Индексы аргументов:
    //
    //args[0] - Сама команда
    //args[1-...] это ваши аргументы команды
    ScriptAPI.sendMessage("TestCommand!");
    ScriptAPI.sendMessage("command: " + args[0]); //test
    ScriptAPI.sendMessage("arg-1: " + args[1]); //первый аргумент
}

Важно! в onCommand_вашакоманда, команда должна быть вся написана с мальенких букв, иначе работать не будет!
