LInjector API Reference

LInjector functions reference.

Unified Naming Convention

⚠️ Since LInjector does not have its own API, we will and you should try to use the ones that support the highest percentage of UNC.

LInjector Custom Functions

Show a Message Box:

messagebox(<string> Title, <string> Content)

Show a Message Box of the Windows System.

Set Rich Presence Status:

setDiscordRPC(<string> Status)

Sets the Rich Presence status to the specified. Only if user has enabled the RPC Status option.

Output to Real Console (rconsole)

Show console and prints the specified. 

rconsoleprint(<string> Content)
-- / Prints to real console as a standard output.

rconsoleinfo(<string> Content)
-- / Print to real console in blue color.

rconsolewarn(<string> Content)
-- / Print to real console in yellow color.

rconsolename(<string> Content)
-- / Sets the real console name to the specified string.

rconsoleerr(<string> Content)
-- / Print to real console in red color.

rconsoleclear(<void>)
-- / Clears the real console.

rconsoleclose(<void>)
-- / Closes the real console.

rconsoleshow(<void>)
-- / Shows the real console.
